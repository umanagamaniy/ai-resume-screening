# 🤖 Uma's AI Assistant

A Spring Boot web application that integrates Google Gemini AI to answer user questions through a clean, beautiful web interface.

## ✨ Features

- 💬 Interactive chat interface to ask AI any question
- 🤖 Google Gemini AI integration
- 🎨 Beautiful gradient UI with smooth interactions
- 💾 H2 in-memory database
- 🔄 Real-time AI responses
- 📱 Clean and responsive design

## 🛠️ Tech Stack

- **Backend:** Java 17, Spring Boot 3.5.15
- **Database:** H2 (in-memory)
- **AI:** Google Gemini API (gemini-2.5-flash-lite)
- **Frontend:** HTML5, CSS3, JavaScript
- **Build:** Maven

## 🚀 Getting Started

### Prerequisites

- Java 17 or higher
- Maven (or use the included Maven wrapper)
- A free Google Gemini API key from [Google AI Studio](https://aistudio.google.com/apikey)
